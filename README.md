# Vulncoz

Vulncoz is a deliberately vulnerable corporate portal designed for
cybersecurity interns and VAPT training.

## Features

- Authentication System
- Employee Directory
- HR Portal
- Ticketing System
- Internal APIs
- Backup Archives
- Information Disclosure Scenarios

## Intended Learning

- Authentication Testing
- Authorization Testing
- API Security
- Information Disclosure
- Secure Code Review
